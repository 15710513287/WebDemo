在创建git项目时勾选中readme选项  将会生成一个readme文件 
也就是项目简介文件  可以在项目首页显示

先在github上复制新建项目的远端地址 
然后本地创建本地版本库  拉取，但是此时没有远端
选择下方管理远端  在远端中的url中输入要提交的远端
添加保存  拉取成功后  再进行提交和推送

直接创建项目之后  复制项目地址进行克隆直接生成本地库

记住密码：
右键  TortoiseGet  设置  git  编辑全局（第二个）
添加
[credential]
  helper = store

如果没有显示贡献记录，检查邮箱是否有添加进项目

基础介绍
http://www.cnblogs.com/edisonchou/p/5990875.html

修改readme文件直接在github上点击这个文件就可以进行修改
readme文件中的排版和html不一样 有自己专门的格式需要注意
